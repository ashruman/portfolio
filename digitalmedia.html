>> Interactive Digital Multimedia

Role: Artist and programmer
Tools used: Javascript, p5.js, Arduino
Team Size: 1

Interactive Digital Multimedia is a course I took on creating interactive art, primarily through programming with JavaScript. I created multiple pieces for critique, almost all of which can be played/experienced in browser.

Log On/Log Off

"Log On/Log Off" is an unwinnable game about trying to clean my room, as well as the first piece I created with p5.js. I programmed multiple scripts and interactive elements, which I then put together to create a simulation of opening windows on a computer.

try anyway?

"try anyway?" is a piece about self-expression, created for the prompt "abstract something that cannot be abstracted". This piece requires microphone and camera access. I wanted to prompt the users/audience to express themselves as many ways as possible, giving them the ability to use text, audio, art, video, and eventually the code itself.

Real Fruit Ninja

The full version of this piece used both p5.js and an Arduino board with a capacitive touch sensor. I created a game similar to "Fruit Ninja", using actual fruit as the controls.
