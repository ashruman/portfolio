>> ValetBike

Role: Backend software engineer
Tools used: Ruby on Rails, MySQL, Git
Team Size: 5
Timeline: 8 weeks

ValetBike is a web application that simulates a bike-rental service, created from scratch as a Software Engineering course project. I was the main contributer for the user account and rental functionality. This required managing and connecting multiple MySQL tables, as well as creating the relevant webpages with Ruby on Rails and HTML.

All the code and more information is available on the project Github repository, and you can also view the live website.
