>> Pygame Visual Novel

Role: Programmer
Tools used: Python, Pygame
Team Size: 1
Timeline: 1 week

After taking an Intro to Programming course during my first year of college, I wanted to try building a game with my new Python programming skills. I created this project on my own in one week, using the Pygame library to create a system that would let me easily create text-based games with simple visuals.

Pygame provides a lot of functionality for creating visuals, sound, and some basic game systems, but I still had to create, customize, or write code for almost every facet of the game. Features include:

  • Selecting a player character.
  • Menus for selecting dialogue, items, or other options.
  • Branching routes depending on dialogue choices.
  • Animated (scrolling) text.
  • Character icons.
  • Scene changes, background music changes, and sound effects can be linked to dialogue and player choices.
  • Inventory system.
  • Title screen.
  • Save files and autosave.
  • Resizable game window.
  • Characters, dialogue, visuals, and audio can be easily changed, allowing for the creation of multiple games/stories using this project as a framework.
